Data description

multi_usv_state.mat:
Historical state data of three USVs, stored as a 1×3 cell array. Each cell corresponds to one USV. For each USV, Row 1 denotes speed, Rows 2–3 denote position coordinates, and Row 4 denotes heading variation.

predicted_dynamic_target_state.mat:
Predicted dynamic target state data generated by the target prediction module.

dynamic_target_angle.mat:
Dynamic target heading history during the simulation.

dynamic_target_position.mat:
Dynamic target position trajectory in the 2D plane during the simulation.

saved_predicted_points.mat:
Saved predicted target points extracted during the simulation for visualization and prediction performance analysis.
